--- title: "Lightsaber battles, but not from Lucas" date: 2006-12-13T12:11:18Z contributors: - Mattie Behrens --- Continuing our coverage of cool Wii Remote hacks, we've found developer Isnoop and a little Mac application called WiiSaber.

WiiSaber is based on his earlier MacSaber program, which used the MacBook Pro's motion sensor to create Star Wars lightsaber sounds when you swung it around. Utilizing code that gets a Mac talking to a Remote, Isnoop has now done the same thing for the Wii Remote, significantly reducing the potential cost should your waving become too exuberant.

You can try a prerelease version of WiiSaber out for yourself, if you have the requisite equipment. Reg Hardware has more information.
