--- title: "Sudoku Fever now available for GBA" date: 2006-02-12T00:21:11Z contributors: - Glen Bayer --- Global Star Sudoku Fever is now available for $19.99 for Game Boy Advance. The game is published by Global star Software and developed by Take-Two Interactive.

The title contains an "infinite number of" Sudoku puzzles, several difficulty levels and 4 grid sizes. The official website for the game can be found here.
