--- title: "N-Sider Chat, Lite edition" date: 2006-06-18T11:03:28Z ---

Every Sunday evening, we will hold a chat on the IRC server irc.opa-ages.com in the #n-sider channel. The easiest way to access the chat is via our N-Forums Java chat room located one click away here.

If you would prefer using IRC and are unsure of how to navigate/acquire the program, start by downloading mIRC, an IRC client. After downloading and installing mIRC, click this link and mIRC should start up. The browser you use may prompt you to launch the application after clicking the link. If so, press the "Launch Application" button if you are using Firefox. When mIRC prompts you with a "Link Request" window, click OK and you will be connected!

If you miss Sunday evening's chat, don't worry. The chat room is always open, and you can often find one or two N-Siders hanging out there. Hope to see you there!
