--- title: "Several new titles announced for GBA and GCN" date: 2004-08-09T16:05:04Z contributors: - Glen Bayer ---

Nintendo Japan has announced several new titles for the GameCube and Game Boy Advance.

Kururin Squash for GameCube is a sequel to the Game Boy Advance puzzle game Kuru Kuru Kururin and is scheduled for an October 14 release in Japan.

F-Zero Climax for Game Boy Advance is scheduled for release this fall in Japan and will support the GBA Wireless Adapter and feature a new Course Edit mode.

Custom Robo for the Game Boy Advance is a brand new sequel to the series that will contain three main robot types including N-Type (standard), S-Type (speed), and P-Type (power).

Square Enix's Kingdom Hearts: Chain of Memories for Game Boy Advance is scheduled for a fall Japanese release.

Banpresto's Super Robot Taisen for GameCube is set for a winter 2004 release in Japan and will feature connectivity with the Game Boy Advance title Super Robot Taisen Original Generation 2.

Fire Emblem: Seima no Kouseki for Game Boy Advance is shaping up for its October release in Japan and will give players the opportunity to play as one of two heroes.
