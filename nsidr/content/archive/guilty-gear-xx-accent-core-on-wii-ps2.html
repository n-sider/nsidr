--- title: "Guilty Gear XX Accent Core on Wii, PS2" date: 2007-06-26T09:58:19Z contributors: - Jon Rust --- Publisher Aksys Games has announced that it is bringing the latest iteration of Arc System Works' Guilty Gear series, Guilty Gear XX Accent Core, to the Playstation 2 and Wii in North America. With the announcement came some new screenshots and concept art. This marks the second time the franchise has appeared on a Nintendo console—the first being Guilty Gear Dust Strikers for the DS.

The latest entry in the series will feature a reworked fighting system, new stages, and two new characters never seen outside of Japan. With the two new characters, A.B.A and Order-Sol, the roster reaches over 20 unique characters, as well as alternate EX versions for each character. The new Generations Mode allows players to set up matches between the old and new characters.

The Wii version of Accent Core utilizes the Wii remote for a brand new control system using simple gestures, but for purists the option is there to use either a GameCube or Classic controller.

Guilty Gear XX Accent Core will be released July 26, 2007 in Japan. A fall release is planned for North America.
