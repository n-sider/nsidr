--- title: "Wi-Fi USB Connector software update (updated)" date: 2006-01-05T11:13:42Z contributors: - Mattie Behrens ---

Nintendo of Europe has released version 1.02 of the driver software for the Nintendo Wi-Fi USB Connector. The update reportedly contains fixes for:

  • The error that occurred in the installation of the version with drivers already installed
  • The problem in which an application error occurred when the registration tool was booted
  • The problem in which the CPU load was quite high when the adapter was not connected

NOA has yet, to our knowledge, to offer this update to customers stateside, but it appears that the update works for all regions. If you find out otherwise, please let us know.

Updated: Some users are reportedly having serious trouble with the update, though we can't confirm whether the problem is isolated or not. The best advice that can be given is this: with any software update, if it ain't broke, don't try fixing it.
