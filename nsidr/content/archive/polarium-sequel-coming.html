--- title: "Polarium sequel coming" date: 2005-08-10T01:45:33Z contributors: - Glen Bayer --- Nintendo is developing a sequel to Polarium for the Nintendo DS, according to Eurogamer.net. The game will be called Polarium: Tsuukin Hito Fude and will feature all the same updates included in the recently announced Game Boy Advance version. This includes 365 new puzzles, a new Time Attack mode as well as allowing players to add color to the game pieces. Polarium: Tsuukin Hito Fude for both GBA and NDS is scheduled for a Japanese release on October 13 for 2,800 yen.
