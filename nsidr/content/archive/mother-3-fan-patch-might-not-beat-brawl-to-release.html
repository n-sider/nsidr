--- title: "Mother 3 fan patch might not beat Brawl to release" date: 2007-10-02T14:36:31Z contributors: - Mattie Behrens --- The Mother 3 Fan Translation project has released a menu patch for the Japanese release of Mother 3.

  • Brawl will "ABSOLUTELY, UNQUESTIONABLY spoil MOTHER 3"
  • The group had set a tentative goal to release a patch before Brawl's release, but is running out of time
  • The menu patch translates menus and items but leaves the story text untouched—players will have to use a walkthrough
  • The team continues to work on the complete translation patch and will release it when it's completely done
