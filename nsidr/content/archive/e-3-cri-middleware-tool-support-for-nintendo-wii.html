--- title: "[E3] CRI Middleware tool support for Nintendo Wii" date: 2006-05-08T17:27:22Z contributors: - Glen Bayer ---

CRI announced its line of middleware tools including ADX, Sofdec, Sound Factory, and ROFS will be available for Nintendo Wii developers. Sofdec allows for multiple movie streams at 60 frames-per-second while ADX enables 16 simultaneous audio streams. Sound Factory is CRI's sound authoring system and ROFS is a virtual file system. All four development libraries are currently available for other game systems. More information on the company's development tools can be found at its official website.
