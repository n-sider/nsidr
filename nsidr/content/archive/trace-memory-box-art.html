--- title: "Trace Memory Box Art" date: 2005-08-02T15:59:07Z contributors: - Colin Castro ---

The box art for the American version of Trace Memory was released a few days ago. The cover depicts the main character holding her arm out to some unknown thing, very similar to other versions of the box art.

For those who are unawares, Trace Memory is an upcoming Nintendo DS game that features Ashley who explores Blood Edward Island in a fashion similar to adventure games of old, like The Dig and Full Throttle.

The game is set for an August 26 release.
