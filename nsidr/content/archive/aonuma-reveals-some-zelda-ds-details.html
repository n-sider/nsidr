--- title: "Aonuma reveals some Zelda DS details" date: 2005-10-10T21:34:22Z contributors: - Mattie Behrens ---

Zelda series producer Eiji Aonuma recently sat down with EGM and revealed a few cryptic clues about the upcoming Zelda DS game to the magazine.

"What I can tell you is that it's going to use the touch screen for a very unique control system that people have never experienced in a Zelda game before.

"What I I think one of the important elements of the Zelda series has always been that you feel like you're in the world doing the things that Link is doing and touching the things that Link is touching.

"This time you'll be able to use the touch screen and actually touch those items and objects yourself.
