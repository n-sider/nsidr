--- title: "Opera Software releases documents for making Wii-friendly sites" date: 2007-06-07T00:00:38Z contributors: - Michael Foster --- Opera Software, creators of the Opera web browser for Nintendo Wii, have released details on how web developers can make their sites Wii-friendly.

The documentation, found on the developer's site for Opera, gives information on how to properly code, utilize images, and use plug-ins that will work well with the Wii Remote and the Wii browser. The Opera browser, powering Wii's Internet Channel, is the exclusive browser for Wii and can be found in the Wii Shop Channel.
