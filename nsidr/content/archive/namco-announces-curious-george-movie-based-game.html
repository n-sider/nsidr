--- title: "Namco announces Curious George movie-based game" date: 2005-10-24T14:54:30Z contributors: - Mattie Behrens ---

Namco Hometek Inc. announced today that it is working on a game based on the upcoming Universal Pictures/Imagine Entertainment animated feature film, Curious George.

The game is currently slated for a February 2006 release on GameCube and Game Boy Advance.
